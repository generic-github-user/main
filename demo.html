Micronotes is a compact JavaScript plugin that allows for beautiful{note}According to me.{/note}, clean, responsive footnotes to be added to plain HTML text. All you need to do is add a reference to the script{note}You can either download the script or add a reference to a hosted version of the script.{/note} somewhere in your HTML. Footnotes can be used for citations{note}github.com/generic-github-user/micronotes{/note}, additional information{note}Did you know that Post It notes were invented by accident?{/note}, or anything else you want.
